# Περιγραφή Προβλήματος

Τα καταφύγια ζώων συχνά αντιμετωπίζουν δυσκολίες στη διαχείριση των ζώων προς υιοθεσία και στην επικοινωνία με τους ενδιαφερόμενους υιοθέτες.

Χωρίς ένα οργανωμένο πληροφοριακό σύστημα μπορεί να υπάρχουν προβλήματα όπως:

- ελλιπής πληροφορία για τα ζώα
- δυσκολία στην αναζήτηση
- μη οργανωμένες αιτήσεις υιοθεσίας
- δυσκολία παρακολούθησης της διαδικασίας υιοθεσίας

Το project προτείνει την ανάπτυξη μιας πλατφόρμας που οργανώνει και απλοποιεί αυτή τη διαδικασία.

---

# Στόχος του Συστήματος

Ο στόχος είναι η ανάπτυξη ενός πληροφοριακού συστήματος που επιτρέπει:

- στα καταφύγια να δημοσιεύουν ζώα προς υιοθεσία
- στους χρήστες να αναζητούν ζώα
- στους χρήστες να υποβάλλουν αιτήσεις υιοθεσίας
- στα καταφύγια να διαχειρίζονται τις αιτήσεις

---

# Ενδιαφερόμενα Μέρη (Stakeholders)

### Πρωτεύοντα

- καταφύγια ζώων
- χρήστες / υιοθέτες
- διαχειριστές συστήματος

### Δευτερεύοντα

- εθελοντές
- τεχνική ομάδα
- διδάσκοντες μαθήματος

---

# Λειτουργικές Απαιτήσεις

Το σύστημα πρέπει να υποστηρίζει:

- δημιουργία λογαριασμού χρήστη
- σύνδεση χρήστη (login)
- δημιουργία προφίλ ζώου
- αναζήτηση ζώων
- φίλτρα αναζήτησης
- υποβολή αίτησης υιοθεσίας
- διαχείριση αιτήσεων από καταφύγια

---

# Μη Λειτουργικές Απαιτήσεις

Το σύστημα πρέπει να είναι:

- εύχρηστο
- επεκτάσιμο
- ασφαλές για βασική διαχείριση χρηστών
- καλά τεκμηριωμένο
- εύκολο στη συντήρηση

---

# Κύριες Οντότητες Συστήματος

Οι βασικές οντότητες του συστήματος μπορεί να είναι:

- User
- Pet
- Shelter
- AdoptionRequest
- PetImage
